# Mini Admin Panel (Flask)

Minimal Flask-based admin panel template for internal tools and small business dashboards.

## Features
- Simple login system
- Session-based authentication
- Protected dashboard route
- Clean project structure

## Typical use cases
- Internal admin tools
- Small business dashboards
- Management panels
- Prototype backends

## Tech stack
- Python
- Flask

## How to run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the app:
   python app.py
3. Open in browser:
   http://127.0.0.1:5000

## Notes
This project is a minimal template designed to be extended
with databases, roles, file uploads or API integrations.
